## Описание 
- Создание одностраничного приложения списка дел с возможностью просмотра, добавления, удаления, копирования и редактирования задач.
- Ввод текста задачи при создании новой задачи и сохранение его в локальном хранилище.
- Добавление кнопки "Удалить" для удаления задачи из списка.
- Копирование задачи при нажатии на кнопку "Копировать" и сохранение копии в начале списка задач.
- Возможность редактирования текста задачи при нажатии на кнопку "Редактировать".
- Сохранение изменений текста задачи при обновлении страницы без потери данных.
###  Технические детали
- Работа с шаблонами (template)
- Использование localStorage для сохранения данных
- Манипуляции с DOM: append, prepend, remove
- Обработка событий: submit, click, blu
- Использование JSON.stringify() и JSON.parse() для работы с localStorage
